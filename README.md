# 🏎️ F1 Driver Performance Analysis (PySpark)

## 📌 Overview

This project analyzes Formula 1 driver performance using PySpark.

Rather than relying only on points or wins (which are highly influenced by the car), the analysis focuses on comparing teammates to better isolate individual driver performance.

---

## 🎯 Objectives

- Build a clean dataset from multiple sources  
- Compare drivers within the same team  
- Create performance indicators  
- Apply PySpark transformations  
- Optimize computations
